Android App

Perfect for shop counters and mobile use

Requirements
  • Android 7.0 or higher
  • 50MB free space
  • Internet for updates
  • Bluetooth for printers
File Details

Format: APK

Size: 86 MB

Version: v7.0.1

Installation Guide
  1. Download the APK file
  2. Enable "Install from unknown sources" in Settings → Security
  3. Open the downloaded APK file
  4. Click "Install" and wait for completion
  5. Open the app and start managing your shop!
Note: Some devices may require additional permissions for printing.

Windows App

Desktop version for shop management

Requirements
  • Windows 10 or 11
  • 4GB RAM minimum
  • .NET 8.0 Runtime
  • 100MB free space
File Details

Format: rar (Portable)

Size: 491 MB

Version: v7.0.1

Installation Guide
  1. Download the rar file
  2. Extract to any folder (e.g., C:\CakeByHtoo)
  3. Run "CakeByHtoo.exe"
  4. No installation needed - it's portable!
  5. Create desktop shortcut if needed
